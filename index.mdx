## What is varg?

varg is an AI video generation SDK that lets you create videos using JSX syntax. Write declarative code like React components, and varg handles AI generation, caching, and video rendering.

```tsx
import { render, Render, Clip, Image, Video } from "vargai/react";
import { fal } from "vargai/ai";

const cat = Image({ prompt: "cute orange cat, Pixar style", aspectRatio: "9:16" });

await render(
  <Render width={1080} height={1920}>
    <Clip duration={3}>
      <Video
        prompt={{ text: "cat waves hello, bounces happily", images: [cat] }}
        model={fal.videoModel("kling-v2.5")}
      />
    </Clip>
  </Render>,
  { output: "hello.mp4" }
);
```

Run it:

```bash
bun run hello.tsx
```

## Quick Start

<Steps>
  <Step title="Install">
    ```bash
    bun install vargai ai
    ```
  </Step>
  <Step title="Set API Key">
    ```bash
    export FAL_KEY=fal_xxx
    ```
    Get your key at [fal.ai/dashboard/keys](https://fal.ai/dashboard/keys)
  </Step>
  <Step title="Create Video">
    ```bash
    bunx vargai hello
    bun run hello.tsx
    ```
  </Step>
</Steps>

## Why varg?

<CardGroup cols={2}>
  <Card title="Declarative JSX" icon="code">
    Write videos like React components. Compose scenes, add transitions, layer audio.
  </Card>
  <Card title="AI-Native" icon="wand-magic-sparkles">
    Built-in support for image generation, video synthesis, voice, music, and lipsync.
  </Card>
  <Card title="Automatic Caching" icon="bolt">
    Same props = instant cache hit. Re-render videos without re-generating AI assets.

  </Card>

## Use Cases

- **Social Media Content**: TikTok, Reels, Shorts at scale
- **Talking Characters**: AI avatars with lipsync and captions
- **Product Videos**: E-commerce showcases, before/after transformations
- **Slideshows**: Image sequences with transitions and music
